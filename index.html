Introducción a Flexbox

¿Qué es Flexbox?

Flexbox es un módulo de diseño incluido en CSS3 que se creó para mejorar el diseño responsive, es decir, para que el tamaño y el posicionamiento de los elementos pueda adaptarse a cada tamaño de pantalla.

¿Cómo se utiliza?

Se le asigna la propiedad "display:flex" al contenedor de los elementos a posicionar, este contenedor flexible (flex container) sería el padre de los elementos flexibles (flex items) que existan dentro de él.

A tener en cuenta

Cuando añadimos flexbox a nuestro contenedor padre, solamente los hijos inmediatos de este se van a ver afectados, es decir, si dentro de mi contenedor padre tengo otro contenedor con mas elementos, estos últimos no se verán afectados por las propiedades flexbox de su abuelo.

¿Cómo funciona?

Flexbox un sistema de diseño de una dimensión donde los elementos solo pueden posicionarse en filas o columnas. Contiene dos ejes, el eje principal (main axis) y el eje transversal (cross axis). Estos ejes cambian según la dirección de los elementos.

*Nota: el punto de la flecha indica donde empieza y hacia donde se dirige, esto es importante para saber como se va a posicionar cada elemento.

Display


Para poder utilizar Flexbox, siempre es necesario añadir display: flex; al contenedor de los elementos a posicionar

Ejemplo:

display: block;

Flex-direction

Esta propiedad nos permite cambiar nuestro main axis o eje principal. Por defecto nuestro eje principal va a ser horizontal y se va a posicionar de izquierda a derecha, a esto se lo conoce como fila o row. Si quisieramos que sea vertical, sería una columna o column.

También podemos intercambiar el inicio y el final del eje principal con row-reverse y column-reverse

Ejemplo:

display: flex;
flex-direction: row;

Justify-content

Nos permite posicionar los elementos a lo largo del eje principal o main axis.

Ejemplo:

Align-items

Flex-wrap

Cuando añadimos display:flex; al contenedor, por defecto los elementos se van a achicar para encajar en una sola fila, es decir flex-wrap: nowrap; pero si quisieramos que conserven su tamaño establecido, podemos aplicar la propiedad flex-wrap: wrap; y se creará más de una fila.

Ejemplo:

display: flex;
flex-wrap: nowrap;
