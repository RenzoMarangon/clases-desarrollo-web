Flex-box

¿Qué es Flex-box?

Flexbox es una propiedad en css que proporciona una forma eficiente de organizar, alinear y distribuir elementos dentro de un contenedor, independientemente del tamaño de la pantalla o del dispositivo utilizado para visualizar la página web.

¿Cómo se utiliza?

Se le asigna la propiedad "display:flex" al elemento padre, es decir, al elemento html que contiene los elementos a organizar

Ejemplo:

display: block;

Flex-direction

Flex-direction afecta a la forma en que los elementos flexibles se distribuyen y alinean dentro del contenedor padre.

Ejemplo:

display: flex;
flex-direction: row;

Justify-content

Alinea y distribuye los elementos flexibles a lo largo del eje X dentro de un contenedor padre.

Ejemplo:

display: flex;
Justify-content: flex-start;

Align-items

Alinea y distribuye los elementos flexibles a lo largo del eje Y dentro de un contenedor padre.

Ejemplo:

display: flex;
align-items: stretch;

Flex-wrap

Controla el tamaño de los elementos dentro de un contenedor, es decir, verifica que se mantenga el tamaño establecido, y si el contenedor no tiene espacio, ubica los elementos en una nueva linea

Ejemplo:

display: flex;
flex-wrap: nowrap;
